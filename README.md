# Animal Shelter Dashboard

## Overview

This project is a Python-based dashboard application that connects to a MongoDB database to manage and visualize animal shelter data. It allows users to perform CRUD operations and analyze data through an interactive interface.

## Features

* Full CRUD operations using MongoDB (Create, Read, Update, Delete)
* Backend system implemented with Python
* Interactive dashboard for data visualization
* Filtering and querying animal shelter records
* Structured object-oriented design

## Technologies Used

* Python
* MongoDB
* PyMongo
* Jupyter Notebook

## How It Works

The backend connects to a MongoDB database and performs CRUD operations on animal records. The dashboard retrieves this data and displays it in an interactive format, allowing users to filter, analyze, and visualize the data.
